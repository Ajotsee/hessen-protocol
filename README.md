# Hessen Protocol

Hessen Protocol is a static website for introducing the Hessen reading-club format and sharing related materials.

## Site

- Custom domain: https://hessenprotocol.com
- Hosting: GitHub Pages

## Structure

- `index.html`: main introduction page
- `download.docx`: reading-log template
- `example_plain.pdf`, `example_made.pdf`: sample reading-log PDFs
- `pdfs/`: member reading-log archive
- `audio.m4a`: audio guide

## Publishing Principle

GitHub Pages serves the files in this repository as a website. The `CNAME` file tells GitHub Pages which custom domain should point here.
